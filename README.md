# Java Full Stack Engineer with Angular - Cognizant Training Program

## Course Overview

Welcome to the Cognizant Java Full Stack Engineer with Angular training program! This comprehensive 13-week course is designed to transform you into a proficient full-stack developer capable of building modern, scalable enterprise applications.

### Program Information
- **Duration**: 13 weeks
- **Format**: Instructor-led training with hands-on projects

## What You'll Learn

This program covers the complete technology stack required for modern enterprise application development:

- **Frontend**: HTML5, CSS3, JavaScript, TypeScript, Angular
- **Backend**: Java 21, Spring Framework, Spring Boot, REST APIs, Microservices
- **Database**: MySQL, JDBC, Spring Data JPA, Hibernate
- **Testing**: JUnit, Mockito, Test-Driven Development
- **DevOps**: Git, Docker, Kubernetes, CI/CD Pipelines
- **Cloud**: Google Cloud Platform (GCP)
- **Methodologies**: Agile, Scrum

### Progressive Enhancement (HTML Exercises)

HTML exercises follow a 3-version progressive enhancement approach:
- **Version 1**: Raw HTML (semantic structure)
- **Version 2**: HTML + CSS3 (custom styling)
- **Version 3**: HTML + Bootstrap 5 (framework-based)

## Getting Started

1. **Clone this repository**
   ```bash
   git clone <repository-url>
   cd course
   ```

2. **Setup your development environment**
   - Follow the guides in [resources/setup-guides](./resources/setup-guides/)

3. **Start with Week 1**
   - Begin with [Linux Fundamentals](./01-linux/)
   - Review [week-01/faq.md](./week-01/faq.md) for key concepts
   - Practice with [week-01/mcq.md](./week-01/mcq.md)

4. **Submit your work**
   - Complete exercises in `submissions/<module>/exercises/`
   - Follow the submission guidelines in each exercise file
