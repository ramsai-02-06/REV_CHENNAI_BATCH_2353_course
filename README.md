# Java Full Stack Engineer with Angular - Cognizant Training Program

## Course Overview

Welcome to the Cognizant Java Full Stack Engineer with Angular training program! This comprehensive 13-week course is designed to transform you into a proficient full-stack developer capable of building modern, scalable enterprise applications.

### Program Information
- **Duration**: 13 weeks
- **Training Partner**: Revature
- **Client**: Cognizant
- **Format**: Instructor-led training with hands-on projects

## What You'll Learn

This program covers the complete technology stack required for modern enterprise application development:

- **Frontend**: HTML5, CSS3, JavaScript, TypeScript, Angular
- **Backend**: Java 21, Spring Framework, Spring Boot, REST APIs, Microservices
- **Database**: MySQL, JDBC, Spring Data JPA, Hibernate
- **Testing**: JUnit, Mockito, Test-Driven Development
- **DevOps**: Git, Docker, Kubernetes, CI/CD Pipelines
- **Cloud**: Google Cloud Platform (GCP)
- **Methodologies**: Agile, Scrum

## Course Structure

### Week 1: Foundation
- Linux fundamentals
- Git and version control
- HTML5 and CSS3
- Agile methodology

### Week 2: JavaScript Fundamentals
- JavaScript ES6+
- Bootstrap 5
- jQuery

### Week 3: Database & Java Basics
- MySQL database
- Core Java fundamentals
- **Project 1 Starts** (Java + MySQL + JDBC)

### Week 4: Core Java Deep Dive
- Object-Oriented Programming
- Collections and Generics
- Multithreading
- Functional Programming
- Design Patterns

### Week 5: Advanced Java
- Reactive Programming
- Testing with JUnit and Mockito
- Logging frameworks
- Debugging and reverse engineering

### Week 6: Spring Framework
- Spring Core (DI, IoC)
- Spring MVC
- Spring Boot fundamentals
- Reactive programming with WebFlux

### Week 7: Spring Data & REST
- Spring Data JPA
- Hibernate ORM
- RESTful API development
- **Project 2 Starts** (Spring Boot + Angular)

### Week 8: Microservices
- Microservices architecture
- API Gateway and Service Discovery
- Security and authentication
- Fault tolerance and resilience

### Week 9: TypeScript & Angular
- TypeScript fundamentals
- Angular setup and basics
- Components and data binding

### Week 10: Advanced Angular
- Dependency Injection
- Routing and navigation
- RxJS and HTTP client
- Forms and state management

### Week 11: Containerization & Cloud
- Docker architecture
- Kubernetes orchestration
- Google Cloud Platform basics
- **Project 3 Starts** (Full stack + Cloud deployment)

### Week 12: DevOps & Cloud Services
- CI/CD pipelines
- GCP services (Compute, Storage, Networking)
- Cloud deployment strategies

### Week 13: Project Week
- Project 3 completion
- Code review and presentation

## Getting Started

1. **Clone this repository**
   ```bash
   git clone <repository-url>
   cd course
   ```

2. **Setup your development environment**
   - Follow the guides in [resources/setup-guides](./resources/setup-guides/)
   - Install all required software listed above

3. **Start with Week 1**
   - Begin with [Linux Fundamentals](./01-linux/)
   - Follow the sequential order of modules
   - Complete exercises after each topic

4. **Practice regularly**
   - Work on exercises in each module
   - Build small projects alongside learning
   - Participate in code reviews
