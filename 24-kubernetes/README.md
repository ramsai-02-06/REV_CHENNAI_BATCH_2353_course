# Kubernetes

## Overview
Kubernetes is an open-source container orchestration platform for automating deployment, scaling, and management of containerized applications.

## Learning Objectives
By the end of this module, you will understand and be able to apply the key concepts and practices of Kubernetes.

- Kubernetes Networking
- Service Discovery
- Ingress Controllers
- Network Policies
- kubectl CLI Commands
- Basic Operations
