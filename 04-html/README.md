# HTML5

## Overview
HTML5 is the latest version of HyperText Markup Language, the standard for creating web pages. This module covers modern HTML5 features and best practices for building semantic, accessible web content.

## Learning Objectives
- Understand HTML5 document structure
- Create semantic HTML markup
- Work with HTML5 forms and input types
- Use HTML5 APIs (storage, geolocation, etc.)
- Build accessible web content

## Topics Covered

### HTML Foundation
- Overview of HTML
- HTML5 Features
- HTML Tags
- Elements and Attributes
- Inline vs Block Elements

### HTML Forms
- Form Element
- Form Attributes (action, method)
- Input Elements
- Input Types (text, email, password, number, date, etc.)
- Label Element
- Textarea
- Select and Option
- Multi-Select
- Fieldset and Legend
- HTML5 Form Validation
- GET vs POST

### HTML5 Advanced
- Web storage (localStorage, sessionStorage)
- Geolocation
- WebSQL Database
- Introduction to CSS3

## Key Concepts

### Semantic HTML
Use semantic tags for better structure and accessibility:
- `<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`

### HTML5 Input Types
- email, url, tel, number, range, date, time, color, search

### HTML5 APIs
- Local Storage and Session Storage
- Geolocation API
- Drag and Drop
- Canvas API
