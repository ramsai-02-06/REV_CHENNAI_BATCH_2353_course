# Google Cloud Platform (GCP)

## Overview
Google Cloud Platform provides cloud computing services including compute, storage, databases, and machine learning.

## Learning Objectives
By the end of this module, you will understand and be able to apply the key concepts and practices of Google Cloud Platform (GCP).
