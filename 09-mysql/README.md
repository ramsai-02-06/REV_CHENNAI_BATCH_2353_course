# MySQL Database

## Overview
MySQL is a popular relational database management system. This module covers SQL fundamentals, database design, and query optimization.

## Learning Objectives
By the end of this module, you will understand and be able to apply the key concepts and practices of MySQL Database.

## Topics Covered

- What Is A Database
- What Is SQL
- Consistency
- Introduction To RDBMS
- Schema
- Table Structure
- SQL Data Types
- Normalization
- Multiplicity
- Data Modeling And ERD
- DDL (CREATE, DROP, TRUNCATE)
- DML (INSERT, UPDATE, DELETE)
- DQL (SELECT)
- DCL (GRANT, REVOKE)
- TCL (COMMIT, ROLLBACK, SAVEPOINT)
- Constraints (PRIMARY KEY, FOREIGN KEY, UNIQUE, NOT NULL, CHECK, DEFAULT)
- Auto Incrementing
- CHECK
- DEFAULT
- CASCADE
- Joins (INNER, LEFT, RIGHT, OUTER, CROSS, SELF, EQUI AND THETA)
- Subquery
- Clauses (WHERE, GROUP BY, HAVING, ORDER BY)
- Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)
- Queries (Complex SQL queries)
