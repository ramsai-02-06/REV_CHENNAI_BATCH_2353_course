# Cheat Sheets

Quick reference guides for all technologies covered in the Java FSE with Angular training program.

## Available Cheat Sheets

### 1. [Linux Commands](./linux-commands.md)
Essential Linux commands for developers including navigation, file operations, process management, and system administration.

### 2. [Git Commands](./git-commands.md)
Complete Git reference covering repository management, branching, merging, remote operations, and collaboration workflows.

### 3. [SQL Queries](./sql-queries.md)
Comprehensive SQL reference including DDL, DML, joins, subqueries, aggregations, and window functions.

### 4. [Java Essentials](./java-essentials.md)
Core Java concepts, syntax, OOP, collections, streams, exception handling, and common patterns.

### 5. [Spring Boot](./spring-boot.md)
Spring Boot essentials including annotations, REST controllers, JPA, configuration, testing, and best practices.

### 6. [Docker Commands](./docker-commands.md)
Docker fundamentals, Dockerfile syntax, container management, Docker Compose, and deployment strategies.

### 7. [Kubernetes kubectl](./kubernetes-kubectl.md)
Kubernetes command reference covering pods, deployments, services, and cluster management.

### 8. [Angular Basics](./angular-basics.md)
Angular framework essentials including components, services, routing, forms, and HTTP client.

### 9. [HTTP Status Codes](./http-status-codes.md)
Complete HTTP status code reference with use cases and Spring Boot examples.

### 10. [REST API Best Practices](./rest-api-best-practices.md)
REST API design principles, naming conventions, versioning, security, and error handling.

## How to Use

### Quick Reference During Development
Keep these cheat sheets open in a browser tab or printed nearby while coding. They provide quick answers without needing to search documentation.

### Learning and Review
Use these as study materials before assessments. Each cheat sheet consolidates key concepts from its respective module.

### Interview Preparation
Review these cheat sheets before technical interviews to refresh your knowledge of key commands and concepts.
